19th Century Sentiment Analysis

Python, NLTK, d3.js

Aggregated data from thousands of 19th and early 20th century newspapers using the Chronicling America API. Used two techniques, sentiment analysis and counting negations, to measure the attitude of publications towards slavery over time and across states. Geocoded the results by state using newspaper locations. Full demo available here. The work was done with a partner for the Computational Journalism class at Georgia Tech.

Developed a hybrid application with the Ionic Framework (based on AngularJS) for an existing social network known as WorkReadyGrad. Relevant Image Finder

Python (Flask), Heroku, various APIs

Developed a Chrome extension to read a body of text and return images relevant to that passage. Used named entity recognition to tag parts of the text, such as People and Locations. Used Google Image Search API and Kairos API for face detection when People were found in the selection. Deployed a Heroku server running Flask to handle requests. Extension available here, though the server is no longer running. This work was done in a team of three for the Computational Journalism class at Georgia Tech.
