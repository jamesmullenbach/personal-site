Image profile intensity extractor

While in Professor Curtis' Cell Physics lab, I developed this GUI in Matlab to satisfy a niche requirement of one of the research projects. It allows users to input any series of images and draw a line in any direction. The program would then find the intensity of the pixels along that line for each image, and output various graphs.
